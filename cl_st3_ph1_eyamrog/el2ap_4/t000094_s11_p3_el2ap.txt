The multimodal text selected communicates through the way the writer
and the illustrator select the visual forms and combine them
with the written text. This combination creates a multimodal text
according to the cultural conventions that surround the text, since
every text is inside a context of culture that determines
the principles for a text being created in that way,
i.e., multimodal texts in Western cultures are different from those
texts in Eastern cultures. In this sense, when dealing with
a multimodal text, it does not make sense to analyse
each resource (music, image, language, etc.) independently, since all these
elements create a whole. Consequently, we talk about the use
